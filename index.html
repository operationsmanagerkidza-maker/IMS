⚙️ Quick Setup — 4 Steps

Open this HTML file in a text editor and fill in your config details at the top.

1
Get Firebase Config

Go to console.firebase.google.com → Your Project → ⚙️ Project Settings → Your Apps → Web App → Copy the firebaseConfig values (no storageBucket needed) and paste them in the FIREBASE_CONFIG block at the top.

2
Enable Realtime Database

Firebase Console → Build → Realtime Database → Create Database → Start in test mode → Choose region → Enable.

3
Set Up Google Drive for Photos

Go to console.cloud.google.com → Enable Google Drive API → Create OAuth 2.0 Client ID (Web) → Add your GitHub Pages URL to Authorized JS origins → Create a shared Drive folder → Paste CLIENT_ID and DRIVE_FOLDER_ID in GDRIVE_CONFIG at the top.

4
Deploy to GitHub Pages

Rename file to index.html → push to GitHub repo → Settings → Pages → Deploy from main branch. The live URL is your app URL.
